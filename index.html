How to Embed Trello

This is a quick demonstration of how to embed a Trello board in your website. It is not a difficult query to find, but it may not work without all the following steps.

  1. Create a board
  2. Make the board public
  3. Copy link to said board
  4. Post the booard in website

That is the TL;DR version.

Detailed Explanaion

Create a board

Click Create New Board.

Make the board public

Remember to make the board public.

Copy link to said board

Click Show Menu > then click More > then find and copy link at bottom of section.
If you have this done everything correctly there will be a message under the link.

Post the board to website

This is not the only way to add Trello to your website, but it is a way that works.

Finished Version

The height and width should to be styled as needed. The initial version is about the size of a single Trello card.
